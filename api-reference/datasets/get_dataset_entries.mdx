Retrieve entries from a specific dataset.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="name" type="string" required="true">

</ParamField>

```bash cURL
curl --request GET \
```

```python Python
import requests

url = "https://api.unify.ai/v0/dataset/my_dataset?limit=None&offset=None"

print(response.text)
```
