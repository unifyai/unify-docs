---
title: 'Rename Dataset'
api: 'PATCH /v0/dataset/{name}'
---
Rename an existing dataset.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="name" type="string" required="true">
Current dataset name (can include forward slashes)
</ParamField>

#### Body

<ParamField body="name" type="string" required="true">
New name of the dataset.
</ParamField>

<RequestExample>

```bash cURL
curl --request PATCH \
  --url 'https://api.unify.ai/v0/dataset/my_dataset' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "name": "renamed-dataset"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/dataset/my_dataset"

headers = {"Authorization": "Bearer <token>"}

json_input = {"name": "renamed-dataset"}

response = requests.request("PATCH", url, json=json_input, headers=headers)

print(response.text)
```
