---
title: "Generate an assistant profile photo from text"
api: 'POST /v0/assistant/photo/generate'
---
Generates a new photo using a text prompt via Replicate and returns the image URL. This action will deduct credits.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="prompt" type="string" required="true">
Text prompt for image generation.
</ParamField>

<ParamField body="aspect_ratio" type="string | null"  default="1:1">
Aspect ratio of the generated image.
</ParamField>

<ParamField body="output_format" type="string | null"  default="webp">
Format of the output image.
</ParamField>

<ParamField body="output_quality" type="integer | null"  default="80">
Quality of the output image (1-100).
</ParamField>

<ParamField body="safety_tolerance" type="number | null"  default="2.0">
Safety tolerance for generation.
</ParamField>

<ParamField body="prompt_upsampling" type="boolean | null"  default="true">
Whether to use prompt upsampling.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \

  --header "Authorization: Bearer $UNIFY_KEY" \
```
