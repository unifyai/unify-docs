---
title: 'Create a new assistant'
api: 'POST /v0/assistant'
---
Creates a new assistant for the authenticated user with the specified configuration.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="first_name" type="string | null" >
First name of the assistant
</ParamField>

<ParamField body="surname" type="string | null" >
Surname of the assistant
</ParamField>

<ParamField body="age" type="integer | null" >
Age of the assistant
</ParamField>

<ParamField body="weekly_limit" type="number | null" >
Weekly time limit for the assistant in hours
</ParamField>

<ParamField body="max_parallel" type="integer | null" >
Maximum number of parallel tasks the assistant can handle
</ParamField>

<ParamField body="region" type="string | null" >
Geographic region of the assistant
</ParamField>

<ParamField body="profile_photo" type="string | null" >
URL to the assistant's profile photo
</ParamField>

<ParamField body="about" type="string | null" >
Brief description about the assistant
</ParamField>

<ParamField body="phone" type="string | null" >
Contact phone number for the assistant
</ParamField>

<ParamField body="email" type="string | null" >
Email address for the assistant
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/assistant' \
  --header "Authorization: Bearer $UNIFY_KEY" \
  --header 'Content-Type: application/json' \
  --data '{
    "first_name": "Ada",
    "surname": "Lovelace",
    "age": 28,
    "weekly_limit": 15.75,
    "max_parallel": 2,
    "region": "North America",
    "profile_photo": "https://example.com/photos/ada.jpg",
    "about": "Mathematician and writer known for work on Analytical Engine",
    "phone": "+1-555-123-4567",
    "email": "ada.lovelace@example.com"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant"

headers = {"Authorization": "Bearer <token>"}

json_input = {"first_name": "Ada", "surname": "Lovelace", "age": 28, "weekly_limit": 15.75, "max_parallel": 2, "region": "North America", "profile_photo": "https://example.com/photos/ada.jpg", "about": "Mathematician and writer known for work on Analytical Engine", "phone": "+1-555-123-4567", "email": "ada.lovelace@example.com"}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```
