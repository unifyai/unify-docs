---
title: "Delete Context Artifact"
api: 'DELETE /v0/project/{project}/contexts/{context}/artifacts/{key}'
---
Deletes an artifact from a context within a project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="project" type="string" required="true">
Name of the project containing the context.
</ParamField>

<ParamField query="context" type="string" required="true">
Name of the context to delete an artifact from.
</ParamField>

<ParamField query="key" type="string" required="true">
Key of the artifact to delete.
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/project/eval-project/contexts/training/artifacts/context-description' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/project/eval-project/contexts/training/artifacts/context-description"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```

</RequestExample>
