---
title: "Update assistant configuration"
api: 'PATCH /v0/assistant/{assistant_id}/config'
---
Updates the configuration parameters of an existing assistant. Profile photo cannot be updated via this endpoint.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="assistant_id" type="integer" required="true">

</ParamField>

#### Body

<ParamField body="weekly_limit" type="number | null" >
Weekly time limit for the assistant in hours
</ParamField>

<ParamField body="max_parallel" type="integer | null" >
Maximum number of parallel tasks the assistant can handle
</ParamField>

<ParamField body="profile_photo" type="string | null" >
URL to the assistant's profile photo
</ParamField>

<ParamField body="profile_video" type="string | null" >
URL to the assistant's profile video
</ParamField>

<ParamField body="about" type="string | null" >
Brief description about the assistant
</ParamField>

<ParamField body="user_phone" type="string | null" >
Contact phone number of the user
</ParamField>

<ParamField body="phone" type="string | null" >
Contact phone number for the assistant
</ParamField>

<ParamField body="email" type="string | null" >
Email address for the assistant
</ParamField>

<ParamField body="user_whatsapp_number" type="string | null" >
WhatsApp number of the user
</ParamField>

<ParamField body="voice_id" type="string | null" >
Id of the voice (Cartesia ID) to use for the assistant
</ParamField>

<ParamField body="country" type="string | null" >
Country code for phone number provisioning (e.g., US, GB)
</ParamField>

<ParamField body="create_infra" type="boolean | null"  default="true">
Whether to create infrastructure for the assistant during update (e.g., phone, email). Set to false for testing.
</ParamField>

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/<assistant_id>/config"

json_input = {"weekly_limit": 20.5, "max_parallel": 3, "profile_photo": "https://example.com/photos/ada.jpg", "profile_video": "https://example.com/videos/ada_new.mp4", "about": "Award-winning mathematician specializing in algorithm development", "user_phone": "+15551234567", "phone": "+15559876543", "email": "ada.lovelace@newdomain.com", "user_whatsapp_number": "+15559876543", "voice_id": "bf0a246a-8642-498a-9950-80c35e9276b5", "country": "GB"}

print(response.text)
```
