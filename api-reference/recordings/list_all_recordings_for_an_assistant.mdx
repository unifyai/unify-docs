---
title: "List all recordings for an assistant"
api: 'GET /v0/assistant/{assistant_id}/recordings'
---
Returns a list of all call recordings for the specified assistant.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="assistant_id" type="integer" required="true">

</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/assistant/<assistant_id>/recordings' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/assistant/<assistant_id>/recordings"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
