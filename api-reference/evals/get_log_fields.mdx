---
title: 'Get Log Fields'
api: 'GET /v0/logs/fields'
---
Returns a mapping of fields and their datatypes from a project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="project" type="string" required="true">
Name of the project to get fields for.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/logs/fields?project=eval-project' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/logs/fields?project=eval-project"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```

</RequestExample>
<ResponseExample>

```json 200

    "entries": {

        "col2": "float"
    }
```
