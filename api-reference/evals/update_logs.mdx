---
title: 'Update Logs'
api: 'PUT /v0/logs'
---
Updates multiple logs with the provided entries. Each entry will be either added
or overridden in the specified logs.

A dictionary of "explicit_types" can be passed as part of the `entries`.
If present, it will override the inferred type of any matching key in all logs.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="ids" type="[integer]" required="true">
List of log IDs to update with new or overriding entries.
</ParamField>

<ParamField body="parameters" type="object"  default={{}}>
Dictionary of key-value parameter pairs to add or update in the logs.
</ParamField>

<ParamField body="entries" type="object"  default={{}}>
Dictionary of key-value entry pairs to add or update in the logs.
</ParamField>

<ParamField body="overwrite" type="boolean"  default={false}>
Whether to overwrite existing logs
</ParamField>

<RequestExample>

```bash cURL
curl --request PUT \
  --url 'https://api.unify.ai/v0/logs' \
  --header "Authorization: Bearer $UNIFY_KEY" \
  --header 'Content-Type: application/json' \
  --data '{
    "ids": [
        123,
        456,
        789
    ],
    "parameters": {
        "explicit_types": {
            "system-prompt": "str"
        },
        "function_definition": "...",
        "system-prompt": "..."
    },
    "entries": {
        "explicit_types": {
            "input": "Image"
        },
        "input": "...",
        "score-test-1": "..."
    },
    "overwrite": false
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/logs"

headers = {"Authorization": "Bearer <token>"}

json_input = {"ids": [123, 456, 789], "parameters": {"explicit_types": {"system-prompt": "str"}, "function_definition": "...", "system-prompt": "..."}, "entries": {"explicit_types": {"input": "Image"}, "input": "...", "score-test-1": "..."}, "overwrite": False}

response = requests.request("PUT", url, json=json_input, headers=headers)

print(response.text)
```
