---
title: 'Delete Log Entries'
api: 'DELETE /v0/logs/entry/{entry}'
---
Deletes a specific entry from multiple logs.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="entry" type="string" required="true">
Name of the entry to delete from the given logs.
</ParamField>

#### Body

<ParamField body="ids" type="[integer]" required="true">
List of log IDs to delete the entry from.
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/logs/entry/entry-v0' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "ids": [
        123,
        456,
        789
    ]
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/logs/entry/entry-v0"

headers = {"Authorization": "Bearer <token>"}

json_input = {"ids": [123, 456, 789]}

response = requests.request("DELETE", url, json=json_input, headers=headers)

print(response.text)
```

</RequestExample>
