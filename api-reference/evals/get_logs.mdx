---
title: 'Get Logs'
api: 'GET /v0/logs'
---
Returns a list of filtered entries from a project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="project" type="string" required="true">
Name of the project to get entries from.
</ParamField>

<ParamField query="filter_expr" type="string | null" >
Boolean string to filter entries. TODO: Detailed page.
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/logs?project=eval-project&filter_expr=len(output)%20>%20200%20and%20temperature%20==%200.5' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/logs?project=eval-project&filter_expr=len(output)%20>%20200%20and%20temperature%20==%200.5"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```

</RequestExample>
