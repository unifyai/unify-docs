---
title: "Get Tile Checkpoint"
api: 'GET /v0/tile/checkpoint'
---


#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="tile_id" type="string | null" >
The ID of the tile to get checkpoint for
</ParamField>

<ParamField query="tab_id" type="string | null" >
The tab ID the tile belongs to
</ParamField>

<ParamField query="name" type="string | null" >
The name of the tile to get checkpoint for
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/tile/checkpoint?tile_id=None&tab_id=None&name=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/tile/checkpoint?tile_id=None&tab_id=None&name=None"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
