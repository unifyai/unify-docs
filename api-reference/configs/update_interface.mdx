---
title: 'Update Interface'
api: 'PUT /v0/interfaces/'
---
Update an interface by ID or by project ID and name.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="interface_id" type="string | null" >
The ID of the interface to update
</ParamField>

<ParamField query="project" type="string | null" >
The project ID the interface belongs to
</ParamField>

<ParamField query="name" type="string | null" >
The name of the interface to update
</ParamField>

<ParamField query="checkpoint" type="boolean"  default=false>
Whether this is a checkpoint update (manual save)
</ParamField>
