---
title: 'List Interfaces'
api: 'GET /v0/interfaces/list'
---
List all interfaces for a project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="project" type="string" required="true">
The project ID to list interfaces for
</ParamField>

<ParamField query="checkpoint" type="boolean"  default=false>
Whether to list checkpoint versions (manually saved)
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/interfaces/list?project=None&checkpoint=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/interfaces/list?project=None&checkpoint=None"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
