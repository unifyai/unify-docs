---
title: "Delete Interface"
api: 'DELETE /v0/interfaces/'
---
Delete an interface by ID or by project ID and name.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="interface_id" type="string | null" >
The ID of the interface to delete
</ParamField>

<ParamField query="project" type="string | null" >
The project ID the interface belongs to
</ParamField>

<ParamField query="name" type="string | null" >
The name of the interface to delete
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/interfaces/?interface_id=None&project=None&name=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/interfaces/?interface_id=None&project=None&name=None"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```

</RequestExample>
