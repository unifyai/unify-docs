---
title: "Get Interface Checkpoint"
api: 'GET /v0/interfaces/checkpoint'
---
Get the latest checkpoint (manual save) for an interface by ID or by project ID and name.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="interface_id" type="string | null" >
The ID of the interface to get checkpoint for
</ParamField>

<ParamField query="project" type="string | null" >
The project ID the interface belongs to
</ParamField>

<ParamField query="name" type="string | null" >
The name of the interface to get checkpoint for
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/interfaces/checkpoint?interface_id=None&project=None&name=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/interfaces/checkpoint?interface_id=None&project=None&name=None"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```

</RequestExample>
