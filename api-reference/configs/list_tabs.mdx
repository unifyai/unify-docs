---
title: "List Tabs"
api: 'GET /v0/tab/list'
---
List all tabs for an interface.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="interface_id" type="string" required="true">
The interface ID to list tabs for
</ParamField>

<ParamField query="name" type="string | null" >
Filter tabs by name
</ParamField>

<ParamField query="checkpoint" type="boolean"  default="false">
Whether to list checkpoint tabs (manual save)
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/tab/list?interface_id=None&name=None&checkpoint=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/tab/list?interface_id=None&name=None&checkpoint=None"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
