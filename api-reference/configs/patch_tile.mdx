---
title: 'Patch Tile'
api: 'PATCH /v0/tile/'
---
Partially update a tile by ID or by tab_id and name.

Only the fields included in the request body will be updated.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="tile_id" type="string | null" >
The ID of the tile to patch
</ParamField>

<ParamField query="tab_id" type="string | null" >
The tab ID the tile belongs to
</ParamField>

<ParamField query="name" type="string | null" >
The name of the tile to patch
</ParamField>

<ParamField query="checkpoint" type="boolean" >
Whether this is a checkpoint update (manual save)
</ParamField>

#### Body

<ParamField body="tile_id" type="string | null" >
The ID of the tile to patch
</ParamField>

<ParamField body="tab_id" type="string | null" >
The tab ID the tile belongs to
</ParamField>

<ParamField body="name" type="string | null" >
The name of the tile to patch
</ParamField>

<ParamField body="checkpoint" type="boolean" >
Whether this is a checkpoint update (manual save)
</ParamField>

<RequestExample>

```bash cURL
curl --request PATCH \
  --url 'https://api.unify.ai/v0/tile/?tile_id=None&tab_id=None&name=None&checkpoint=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/tile/?tile_id=None&tab_id=None&name=None&checkpoint=None"

headers = {"Authorization": "Bearer <token>"}

json_input = {}

response = requests.request("PATCH", url, json=json_input, headers=headers)

print(response.text)
```
