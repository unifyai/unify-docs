---
title: "Delete Tab"
api: 'DELETE /v0/tab/'
---
Delete a tab by ID or by interface_id and name.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="tab_id" type="string | null" >
The ID of the tab to delete
</ParamField>

<ParamField query="interface_id" type="string | null" >
The interface ID the tab belongs to
</ParamField>

<ParamField query="name" type="string | null" >
The name of the tab to delete
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/tab/?tab_id=None&interface_id=None&name=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/tab/?tab_id=None&interface_id=None&name=None"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```

</RequestExample>
