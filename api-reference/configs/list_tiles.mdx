---
title: 'List Tiles'
api: 'GET /v0/tile/list'
---
List all tiles for a tab.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Query Parameters

<ParamField query="tab_id" type="string" required="true">
The tab ID to list tiles for
</ParamField>

<ParamField query="name" type="string | null" >
Filter tiles by name
</ParamField>

<ParamField query="type" type="string | null" >
Filter tiles by type
</ParamField>

<ParamField query="checkpoint" type="boolean" >
Whether to list checkpoint tiles (manual save)
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/tile/list?tab_id=None&name=None&type=None&checkpoint=None' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/tile/list?tab_id=None&name=None&type=None&checkpoint=None"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("GET", url, headers=headers)

print(response.text)
```
