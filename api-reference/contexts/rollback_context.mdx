---
title: "Rollback Context"
api: 'POST /v0/project/{project_name}/contexts/{context_name}/rollback'
---
Rolls back a context to a specific version.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="project_name" type="string" required="true">
Name of the project to create context in.
</ParamField>

<ParamField query="context_name" type="string" required="true">
Name of the context to rollback.
</ParamField>

#### Body

<ParamField body="version" type="integer | null" >

</ParamField>

<ParamField body="commit_hash" type="string | null" >

</ParamField>
