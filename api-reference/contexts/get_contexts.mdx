---
title: 'Get Contexts'
api: 'GET /v0/project/{project_name}/contexts'
---
Get a list of contexts within a project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="project_name" type="string" required="true">
Name of the project to create context in.
</ParamField>

#### Body

<ParamField body="name" type="string" required="true">
Context name, can be nested using '/' (e.g., 'parent/child'). Must contain only alphanumeric characters, underscores, and hyphens.
</ParamField>

<ParamField body="description" type="string | null" >
Optional description of the context
</ParamField>

<RequestExample>

```bash cURL
curl --request GET \
  --url 'https://api.unify.ai/v0/project/my_project/contexts' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "name": "experiment1/trial1",
    "description": "Context for experiment 1 trial 1"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/project/my_project/contexts"

headers = {"Authorization": "Bearer <token>"}

json_input = {"name": "experiment1/trial1", "description": "Context for experiment 1 trial 1"}

response = requests.request("GET", url, json=json_input, headers=headers)

print(response.text)
```

</RequestExample>
