---
title: "Rename Context"
api: 'PATCH /v0/project/{project_name}/contexts/{context_name}/rename'
---
Rename an existing context within a project.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="project_name" type="string" required="true">

</ParamField>

<ParamField query="context_name" type="string" required="true">

</ParamField>

#### Body

<ParamField body="name" type="string" required="true">
New name for the context, must meet naming rules
</ParamField>

<RequestExample>

```bash cURL
curl --request PATCH \
  --url 'https://api.unify.ai/v0/project/<project_name>/contexts/<context_name>/rename' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "name": "experiment2/trial3"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/project/<project_name>/contexts/<context_name>/rename"

headers = {"Authorization": "Bearer <token>"}

json_input = {"name": "experiment2/trial3"}

response = requests.request("PATCH", url, json=json_input, headers=headers)

print(response.text)
```

</RequestExample>
