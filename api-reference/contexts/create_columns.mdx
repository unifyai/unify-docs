---
title: 'Create Columns'
api: 'POST /v0/project/{project_name}/contexts/{context_name}/columns'
---
Creates columns in a context within a project. Columns can have explicit types
or their types can be auto-detected when data is added.

The columns are specified as a dictionary mapping column names to their types.
If a type is specified as null, the type will be auto-detected when data is added.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="project_name" type="string" required="true">
Name of the project containing the context.
</ParamField>

<ParamField query="context_name" type="string" required="true">
Name of the context to create columns for.
</ParamField>

#### Body

<ParamField body="columns" type="object" required="true">
Dictionary mapping column names to their explicit types (or None for auto-detection).
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/project/my_project/contexts/my_context/columns' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "columns": {
        "accuracy": "float",
        "model_name": "str"
    }
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/project/my_project/contexts/my_context/columns"

headers = {"Authorization": "Bearer <token>"}

json_input = {"columns": {"accuracy": "float", "model_name": "str"}}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```
