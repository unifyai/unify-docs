---
title: 'Delete Artifact'
api: 'DELETE /v0/dataset/{dataset}/artifacts/{key}'
---
Deletes an artifact from a dataset.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="dataset" type="string" required="true">
Name of the dataset to delete an artifact from.
</ParamField>

<ParamField query="key" type="string" required="true">
Key of the artifact to delete.
</ParamField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://api.unify.ai/v0/dataset/eval-dataset/artifacts/dataset-description' \
  --header "Authorization: Bearer $UNIFY_KEY"
```

```python Python
import requests

url = "https://api.unify.ai/v0/dataset/eval-dataset/artifacts/dataset-description"

headers = {"Authorization": "Bearer <token>"}

response = requests.request("DELETE", url, headers=headers)

print(response.text)
```

</RequestExample>
