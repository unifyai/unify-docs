---
title: 'Create Artifacts'
api: 'POST /v0/dataset/{dataset}/artifacts'
---
Creates one or more artifacts associated to a dataset. Artifacts are
dataset-level metadata that don't depend on other variables.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="dataset" type="string" required="true">
Name of the dataset the artifacts belong to.
</ParamField>

#### Body

<ParamField body="artifacts" type="object" required="true">
Dictionary containing one or more key:value pairs that will be stored as artifacts within a dataset.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/dataset/eval-dataset/artifacts' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{
    "artifacts": {
        "processed": false,
        "traffic": "production"
    }
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/dataset/eval-dataset/artifacts"

headers = {"Authorization": "Bearer <token>"}

json_input = {"artifacts": {"processed": False, "traffic": "production"}}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```

</RequestExample>
