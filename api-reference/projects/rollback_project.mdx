---
title: "Rollback Project"
api: 'POST /v0/project/{project_name}/rollback'
---
Rolls back a project to a specific version.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Path Parameters

<ParamField query="project_name" type="string" required="true">
Project name, may contain slashes
</ParamField>

#### Body

<ParamField body="commit_hash" type="string" required="true">

</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/project/proj/a/rollback' \
  --header "Authorization: Bearer $UNIFY_KEY"
  --header 'Content-Type: application/json' \
  --data '{}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/project/proj/a/rollback"

headers = {"Authorization": "Bearer <token>"}

json_input = {}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```
