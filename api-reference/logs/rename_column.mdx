---
title: 'Rename Column'
api: 'POST /v0/logs/rename_column'
---
Renames a column across all logs in a project. This includes:
- Updating the field type record
- Renaming the column in all logs (regular and history)

The operation is atomic - either all renames succeed or none do.

#### Authorizations

<ParamField header="Authorization" type="string" required="true">
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token.
</ParamField>

#### Body

<ParamField body="project" type="string" required="true">
Name of the project the column belongs to.
</ParamField>

<ParamField body="context" type="string | null"  default={""}>
The context of the column to rename.
</ParamField>

<ParamField body="old_field_name" type="string" required="true">
The current name of the column to rename.
</ParamField>

<ParamField body="new_field_name" type="string" required="true">
The new name for the column.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url 'https://api.unify.ai/v0/logs/rename_column' \
  --header "Authorization: Bearer $UNIFY_KEY" \
  --header 'Content-Type: application/json' \
  --data '{
    "project": "eval-project",
    "context": "test-context",
    "old_field_name": "score",
    "new_field_name": "score_new"
}'
```

```python Python
import requests

url = "https://api.unify.ai/v0/logs/rename_column"

headers = {"Authorization": "Bearer <token>"}

json_input = {"project": "eval-project", "context": "test-context", "old_field_name": "score", "new_field_name": "score_new"}

response = requests.request("POST", url, json=json_input, headers=headers)

print(response.text)
```

</RequestExample>
